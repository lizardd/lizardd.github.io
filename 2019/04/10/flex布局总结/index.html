Flex布局的总结

  1. 布局元素分为box和item，box容纳item并且使item依据自己的想法进行显示box和item是相对而言。相对于item，box是item的父级节点。

  • 要使用flex布局，则需要在容器上标注css

  • flex布局下，box 容器的属性有6个

  1. flex-direction 属性决定主轴方向
    【[row]，row-reverse，column，column-reverse】
  2. flex-wrap item在轴线排不下，如何换行【[nowrap],wrap,wrap-reverse】
  3. flex-flow 是flex-direction和flex-wrap的组合
  4. justify-content 项目在主轴的对齐方向【flex-start，flex-end，center， space-between，space-around】
  5. align-items 交叉轴上如何对齐【flex-start，flex-end，center，baseline，stretch】
  6. align-content 多根轴线对齐方式【flex-start，flex-end，center，space-between，space-around，stretch】
  • 项目属性
  1. order 排列顺序 越小越靠前 默认0
  2. flex-grow 放大比例 默认0 有剩余空间也不放大
  3. flex-shink 缩小比例 默认1 空间不足会缩小
  4. flex-busis 定义了在分配多余空间前，项目占据的主轴空间，默认auto，项目本来大小
  5. flex 是flex-grow，flex-shrink，flex-basis的简写 默认0 1 auto；
  6. align-self 允许项目有和其他项目不一样的对齐方式，可以覆盖align-items，默认auto表示继承父类元素的align-items属性，如果没有父类元素则等同于stretch。
  • css3新单位
  1. vh 可视窗口的高度，假设高度1200px 则 1vh=12px
  2. vw 可视窗口的宽度，假设宽度1200px 则 1vh=12px
  3. vm 相对于视窗的宽度或高度中较小的那个，1vm=较小那个长度/100
